Epsilora API Server

Server Status

This is the backend API server for the Epsilora educational platform. If you're seeing this page, the server is online and responding to requests.

Available Endpoints

GET /health - Check server health
POST /api/auth/login - User login
POST /api/auth/signup - User registration
GET /api/courses - Get user courses
POST /api/generate-quiz - Generate a quiz
